About Me

Hello, I'm Cassidy. You can also call me Cass. I really love working with computers. Learning new technologies and creating something new is one of my favorite things. I'm currently pursuing a degree in Computer Science. I'm studying physics and calculus in my current quarter. I work at the media center for my school helping other students with all kinds of technological tasks covering subjects like photography, printing, computer program help and lots of others. In my spare time I'm enjoying learning the C++ language and building small prototype games with the Godot engine. Stepping away from computers, I love to enjoy the natural wonders of the pacific northwest. The pacific northwest has been home to me for more than 2 decades. Some of my favorite activities include exploring beaches, forests, parks and photographing wildlife. I'm also a huge fan of music one of my dreams is to work as a recording engineer for great bands.

Projects

ioChatroom

A basic multi-user chatroom that allows visitors to create or join specifically named rooms and chat in real time. Project was built using html, css, javascript with a focus on the io library for multi-user support.
This project was taught in a tutorial on YouTube by Dave Gray.
Note: This project is hosted with renders free service which spins down servers during inactivity. For this reason it may take significantly longer to load than normal.

Self Driving Car

A machine learning project in which a car learns to drive itself through repeating attempts at creating a path through traffic then adjusting based on the input information its neural network recieved. This project was taught by Radu Mariescu-Istodor from University of Eastern Finland.

Fancy A Run

An endless runner game playable on desktop web browsers. Developed using Unity WebGL. Hosted on itch.io.
